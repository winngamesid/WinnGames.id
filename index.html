Basic Installation

  1. Instal Steam Tool yang sudah kalian download dan buka Steam Tool
  2. Ekstrak file game yang telah kalian download
  3. Block/pilih file hasil ekstrasi game tersebut
  4. Drag file yang dipilih ke logo Steam Tool yang mengambang di layar
  5. Klik kanan di logo mengambang Steam Tool lalu pilih Restart Steam

NB: Untuk lebih jelasnya tonton video tutorial di atas.

Cara Memasukkan DLC ke dalam game

  1. Buka aplikasi Steam Tool hingga muncul floating windows Steam Tool
  2. Buka browser lalu ketik steamdb.info di kolom pencarian
  3. Di kolom pencarian ketik game yang ingin kalian mainkan
  4. Cari DLC lalu klik
  5. Copy semua tulisan di kolom "Downloadable Content"
  6. Lalu drag ke logo Steam Tool
  7. Klik kanan di logo Steam Tool pilih Restart Steam

NB: Untuk lebih jelasnya tonton video tutorial di atas.

Troubleshooting

Common Issues:

  • Buka File Explorer
  • Buka C:\Program Files(x86)\Steam\config\depotcache
  • Kemudian hapus semua file di dalamnya
  • Buka C:\Program Files(x86)\Steam\config\stplug-in
  • Kemudian hapus semua file di dalamnya
  • Lalu restart Steam kalian
  • Lakukan instalasi game dari awal seperti video tutorial pertama kali

NB: Untuk lebih jelasnya tonton video tutorial di atas.
